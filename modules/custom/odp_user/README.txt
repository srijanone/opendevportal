INTRODUCTION
------------

User module allows users to add user into program. This module has sub module
login redirects which redirect users as per their assigned program and roles.


REQUIREMENTS
------------

The user module just requires the odp_core module.


INSTALLATION
------------

Install as you would normally install a Drupal module.


CONFIGURATION
-------------

 * Visit the opendevx configuration page at Administration » Configuration »
   Opendevx » Login redirect settings.

 * Enter the redirect URL as per required role.
